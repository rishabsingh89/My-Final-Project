🔗 Connect React Frontend

In your React project, create a .env file:

REACT_APP_API_URL=http://localhost:5000/api

Then call APIs like:

// Example: Login from React const res = await fetch(`${process.env.REACT_APP_API_URL}/auth/login`, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ email, password }) }); const data = await res.json(); localStorage.setItem('token', data.token); // Example: Get inventory (authenticated) const inv = await fetch(`${process.env.REACT_APP_API_URL}/inventory`, { headers: { Authorization: `Bearer ${localStorage.getItem('token')}` } });
